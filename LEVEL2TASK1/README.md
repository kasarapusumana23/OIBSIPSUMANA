# 🧮 Calculator Web Application

## 🎯 Project Overview

This project is a responsive Calculator Web Application developed using HTML, CSS, and JavaScript. The application performs basic arithmetic operations through an intuitive and interactive user interface.

The calculator is designed to provide accurate calculations while maintaining a clean, modern, and user-friendly design.

---

## ✨ Features

✅ Addition

✅ Subtraction

✅ Multiplication

✅ Division

✅ Decimal Calculations

✅ Clear (AC) Function

✅ Delete (⌫) Function

✅ Keyboard Input Support

✅ Responsive Design

✅ Interactive User Interface
