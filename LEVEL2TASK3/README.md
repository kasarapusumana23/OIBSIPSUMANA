# 🌸 Daily Bloom – To-Do Web Application

## 🎯 Project Overview

Daily Bloom is a modern and responsive To-Do Web Application designed to help users organize and manage their daily tasks efficiently. The application provides an intuitive interface for adding, editing, deleting, and tracking tasks, making productivity simple and enjoyable.

The project focuses on task management, user interaction, and data persistence using Local Storage.

---

## ✨ Features

✅ Add New Tasks

✅ Edit Existing Tasks

✅ Delete Tasks

✅ Mark Tasks as Completed

✅ Separate Pending & Completed Task Lists

✅ Local Storage Support

✅ Responsive Design

✅ Modern Lavender & Mint Theme

✅ Interactive User Interface

## 🌟 Project Highlights

### 📝 Task Management

Users can create, edit, and delete tasks effortlessly.

### ✅ Progress Tracking

Tasks can be marked as completed and moved to a separate completed section.

### 💾 Data Persistence

Tasks are stored using Local Storage, ensuring data remains available even after refreshing the browser.

Designed with a beautiful Lavender & Mint theme to provide a clean and pleasant user experience.

### 📱 Fully Responsive

Works seamlessly across desktops, tablets, and mobile devices.

## 🌷 Why "Daily Bloom"?

The name **Daily Bloom** represents personal growth and productivity through consistent daily progress. Just as flowers bloom with care and attention, users can achieve their goals by completing tasks one step at a time.
