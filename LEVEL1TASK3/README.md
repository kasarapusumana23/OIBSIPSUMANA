# 🌡️ Temperature Converter Website

## 🎯 Project Overview

This project is a responsive Temperature Converter Website developed using HTML, CSS, and JavaScript. The application allows users to easily convert temperature values between Celsius (°C), Fahrenheit (°F), and Kelvin (K) through a simple and user-friendly interface.

The project demonstrates the implementation of JavaScript logic, user input handling, and responsive web design principles.

## 🔢 Temperature Conversion Formulae

### Celsius to Fahrenheit

```text
(°C × 9/5) + 32
```

### Fahrenheit to Celsius

```text
(°F - 32) × 5/9
```

### Celsius to Kelvin

```text
°C + 273.15
```

### Kelvin to Celsius

```text
K - 273.15
```

### Fahrenheit to Kelvin

```text
(°F - 32) × 5/9 + 273.15
```

### Kelvin to Fahrenheit

```text
(K - 273.15) × 9/5 + 32
```
