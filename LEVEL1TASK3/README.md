# 🌡️ Temperature Converter

## 📖 Description

A simple and responsive Temperature Converter developed using HTML, CSS, and JavaScript. This application allows users to convert temperature values between Celsius, Fahrenheit, and Kelvin quickly and accurately through an intuitive user interface.

---

## ✨ Features

* Convert Celsius to Fahrenheit and Kelvin
* Convert Fahrenheit to Celsius and Kelvin
* Convert Kelvin to Celsius and Fahrenheit
* Real-Time Temperature Conversion
* User-Friendly Interface
* Responsive Design
* Input Validation

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 🔢 Temperature Conversion Formulae

### Celsius to Fahrenheit

(F × 9/5) + 32

### Fahrenheit to Celsius

(F - 32) × 5/9

### Celsius to Kelvin

C + 273.15

### Kelvin to Celsius

K - 273.15
